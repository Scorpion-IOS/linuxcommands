COMMANDS 

Obtaining System Information

Command [option] [argument]

Description

whoami

Returns the current user name

pwd

Finds the current system location

uname

Gets information about the operating system

lsblk

List information about all available block devices.

cat /proc/cpuinfo

Display CPU information from the /proc pseudo filesystem

cat /proc/meminfo

Display the system’s memory information from the /proc pseudo-filesystem.

cat /etc/os-release

Display the operating system’s identification information as provided by the vendor.

Note: The information is located in a dedicated file in the Linux filesystem.

ps -aux

Command to display a user-oriented snapshot of all currently running processes for all system users, including processes that are not attached to a terminal. Make sure the output includes the process owner.

Service  - -status-all

Display the current status of all available services using the

uname -vr

Use the uname command to display only the machine’s kernel version and release


 

Working with files and folders

Command [option] [argument]

Description

pwd

This command displays the present working directory where you are currently in.

ls

This command will list the content of a directory.
  ls -la  _   

This command will list all the content of a directory including the hidden files and directories.

mkdir

This command will create a new directory, provided it doesn't exist.

 mkdir -p  This command will create nested directories
 

rmdir

This command will remove/delete an existing directory, provided it is empty.

cd

This command is used to change the directory.

cd ..

This command will take us one level up the directory tree

touch filename

Expl: $ touch hello.txt  
 This command will creates a new file.

rm filename

This command will delete a file.

rm -f hello.txt

This command forcefully deletes a file

rm -rf example

This command forcefully and recursively deletes a directory along with its content.   
 Be careful while performing delete operation.

 

cp file1 file2

This command copies the content of file file1 into file file2.
 Expl: $ cp hello.txt hi.txt
 

cp -r dir1 dir2

This command copies the content of directory dir1 into directory dir2.

mv

Rename files and directories
 Expl: $ mv hello.txt hi.txt

cat filename

This will print the content of a file.
 $ cat hello.txt 

The quick brown fox jumps over the lazy dog.

head filename

This command will print the first 10 lines of a file.

$ head fruits.txt

tail filename

This command will print the last 10 lines of a file.
 $ tail fruits.txt

tail -f filename

This will print the last 10 lines of a file and will keep printing new lines as they get appended to the file.

$ tail -f log.txt
